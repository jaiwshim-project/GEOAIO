## 2단계: 사유별 즉시 조치 매뉴얼

### A. "발견됨 — 현재 색인되지 않음" 처리
원인: 신생 도메인 신뢰도 부족 + 사이트맵 우선순위 낮음 + 내부 링크 부족

- [ ] Tistory 관리 → SEO·블로그 정보 → "검색엔진 노출 허용" ON 확인
- [ ] `https://digitalsmile.tistory.com/sitemap.xml` 직접 접속 → 50개 URL이 모두 포함되어 있는지 확인
- [ ] Search Console > 사이트맵 → `sitemap.xml` 재제출
- [ ] Search Console > URL 검사 → 미색인 URL 5개 골라 "색인 생성 요청" 직접 클릭 (24~72시간 후 반영)
- [ ] 각 글 본문 끝에 같은 카테고리 다른 글로 가는 내부 링크 3개 이상 추가
- [ ] 카테고리 인덱스 페이지(`/category/임플란트` 등)에서 모든 글이 노출되는지 확인

### B. "크롤링됨 — 현재 색인되지 않음" 처리
원인: 콘텐츠가 양산형으로 보임 / 고유 가치 신호 부족 / E-E-A-T 약함

- [ ] 글당 본문 길이 1,700자 이상인지 (현재 우리 엔진 기본값 부합)
- [ ] **첫 문단에 작성자 권위 신호** 명시 — "박찬익 원장 15년 경력" (우리 엔진 자동 삽입)
- [ ] **글당 고유 데이터 1개 이상** — 자체 케이스 / 대전 지역 데이터 / 자체 통계
- [ ] **외부 인용 1개 이상** — Christian Coachman 인터뷰·논문 등 출처 링크
- [ ] **동일 키워드 글 3편 이상 중복 금지** — DSD 기초 글 10편 → 3편 이내로 축소·통합
- [ ] 이미지 alt 텍스트 작성 (Tistory는 자동 alt 없음)
- [ ] 작성자 소개 페이지(`/about` 또는 카테고리 1번 글)로 모든 글에서 링크

### C. "중복 페이지" 처리
원인: 유사 주제 글 다수, Google이 한 글을 대표로 골라 나머지 무시

- [ ] 카테고리 ① 디지털스마일치과 3편이 모두 DSD 비교·소개·만족도 → **사실상 1편으로 통합 후 나머지 삭제**
- [ ] 신규 240편 매트릭스에서는 **6 카테고리 × 9 intent**로 주제 분산 (`scripts/content-matrix-240.csv` 참고)
- [ ] 같은 카테고리 안에서 유사 제목 3편 이상이 되지 않도록 매트릭스 검수
- [ ] Tistory는 canonical 자동 처리이지만, 비슷한 주제의 두 글에서는 한쪽 글 본문에 다른 글을 명시적으로 링크해 "원본·심화" 관계로 만들 것

## 3단계: 발행 전 체크리스트 (글마다 적용)

새로 발행하는 40편/주 모든 글에 적용.

- [ ] 카테고리 6개 중 1개에 반드시 분류 (미분류 금지)
- [ ] 제목에 카테고리 키워드 + 검색 의도 키워드 포함 (예: "임플란트 비용 — ...")
- [ ] H2 5~7개 중 5개 이상이 질문형 (`~는 어떻게/왜/무엇이?`)
- [ ] 각 H2 시작에 "X란 ~이다" 정의문 (우리 엔진 자동)
- [ ] 각 H2 끝에 "**핵심: ~~이다**" 한 줄 요약 (우리 엔진 자동)
- [ ] FAQ 3개 이상, "Q: ... A: ..." 형식 정확히
- [ ] 3열 마크다운 표 1개 이상
- [ ] 도입부 권위 신호 + 결론 권위 신호 (지역·기간·성과)
- [ ] 본문 내부 링크 3개 (같은 카테고리 다른 글로)
- [ ] 외부 출처 링크 1개 이상 (학회·논문·정부기관·창시자 인터뷰)
- [ ] 이미지 alt + 캡션 작성
- [ ] 해시태그 10개 (Tistory 태그 시스템)
- [ ] 글 발행 직후 Search Console URL 검사 → 색인 생성 요청

## 4단계: 6주 측정 계획

매주 금요일 오후 6시 측정.

| 주차 | 발행 누적 | 색인 목표 | 측정 항목 |
|---|---|---|---|
| 1주 | 40편 | 색인 15편 (37.5%) | 발견됨/크롤링됨 비율 |
| 2주 | 80편 | 색인 35편 (43.7%) | "크롤링됨 — 미색인" 사유 비율 |
| 3주 | 120편 | 색인 60편 (50%) | 평균 색인 소요 시간 |
| 4주 | 160편 | 색인 90편 (56%) | 노출 1회 이상 페이지 수 |
| 5주 | 200편 | 색인 130편 (65%) | 클릭 발생 페이지 |
| 6주 | 240편 | 색인 170편 (70%+) | 브랜드 검색·AI 인용 흔적 |

> **현실 기대치**: 신생 Tistory 서브도메인은 첫 6주 안에 90% 색인 도달이 매우 어렵습니다. 70~75%면 성공 케이스. 90%는 백링크·브랜드 검색이 누적되는 3~6개월차 목표로 잡으세요.

## 5단계: 90% 도달을 위한 추가 신호 (2개월차 이후)

색인율 정체 시 외부 신호 부족이 원인. 추가 작업:

- [ ] **백링크 5~10개** — 지역 카페·블로그·언론 보도 자료
- [ ] **Google 비즈니스 프로필** 의 웹사이트 링크
- [ ] **네이버 블로그·카카오 채널** 에 글 일부 발췌 + 원문 링크
- [ ] **유튜브 동영상** 1편 이상 → 설명란에 글 링크
- [ ] **브랜드 검색량** 월 50~100회 이상 (직접 "디지털스마일치과 대전" 검색)
- [ ] **Wikipedia·나무위키** DSD 항목 외부링크 추가 (난이도 높음)
