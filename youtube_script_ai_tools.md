## [본론1 - 2분] Claude Code Voice Mode

[화면: "01 Claude Code Voice Mode" 타이틀 카드]
[화면: Claude Code 터미널 화면 캡처]

> 첫 번째로 소개할 도구는 Claude Code의 Voice Mode입니다.

> Claude Code는 Anthropic에서 만든 터미널 기반 AI 코딩 어시스턴트인데요, 여기에 음성 모드가 추가되면서 게임 체인저가 됐습니다.

[자막: Claude Code Voice Mode = 음성으로 프롬프트 입력 + AI 응답을 즉시 청취]

> 핵심이 뭐냐면요, 키보드를 치지 않고 목소리만으로 프롬프트를 입력할 수 있어요. 그리고 AI의 응답도 음성으로 바로 들을 수 있습니다.

[화면: Voice Mode 활성화 화면 시연 영상]

> 이게 왜 좋으냐면요, 일단 멀티태스킹이 가능해집니다. 예를 들어 볼게요.

### 데모 시나리오

[화면: 개발자가 화이트보드 앞에서 설계하는 모습 + Claude Code Voice 화면 분할]

> 여러분이 지금 화이트보드에 시스템 아키텍처를 그리고 있다고 해봅시다. 손에는 마커를 들고 있으니까 키보드를 칠 수가 없잖아요.

> 이때 이렇게 말하면 됩니다.

[자막: "이 프로젝트에서 사용자 인증 미들웨어 만들어줘. JWT 기반으로."]

> 그러면 Claude Code가 알아서 코드를 작성하고, 결과를 음성으로 설명해줘요.

[화면: 코드가 생성되는 터미널 애니메이션]

> 브레인스토밍할 때도 정말 유용합니다. 산책하면서, 운전하면서, 운동하면서 아이디어를 AI와 주고받을 수 있어요. 완전한 핸즈프리 대화가 가능한 거죠.

[자막: 핸즈프리 코딩 & 브레인스토밍의 시대]

> 개발자분들, 특히 아이디어는 많은데 타이핑이 귀찮으신 분들 - 이거 꼭 써보세요.

## [본론2 - 2분] Lightning V3

[화면: "02 Lightning V3" 타이틀 카드]
[화면: Lightning V3 공식 페이지 또는 데모 화면]

> 두 번째 도구는 Lightning V3입니다. 이름부터가 번개죠? 이름값 제대로 합니다.

> Lightning V3는 현존하는 가장 가볍고 빠른 TTS, 그러니까 텍스트를 음성으로 바꿔주는 모델인데요.

[자막: TTS = Text-to-Speech, 텍스트 음성 변환]

> 스펙을 보면 입이 딱 벌어집니다.

[화면: 스펙 비교 인포그래픽]
[자막: 지연시간 100ms / WVMOS 3.89 / 15개+ 언어 지원]

> 첫째, 지연 시간이 100밀리초. 0.1초예요. 텍스트 넣자마자 거의 즉시 음성이 나옵니다.

> 둘째, 음성 품질 지표인 WVMOS 점수가 3.89입니다. 이 정도면 사람이 말하는 것과 거의 구분이 안 되는 수준이에요.

> 셋째, 15개 이상의 언어를 지원합니다. 한국어도 당연히 포함이고요.

> 그리고 결정적으로, 이게 최경량 모델이라 리소스를 거의 안 먹어요.

### 데모 시나리오

[화면: Lightning V3 데모 인터페이스 화면 캡처]

> 자, 실제로 어디에 쓸 수 있는지 볼까요?

> 예를 들어 여러분이 교육 콘텐츠를 만든다고 해봅시다. 강의 스크립트를 텍스트로 작성한 다음에 Lightning V3에 넣으면요,

[화면: 텍스트 입력 -> 음성 출력 과정 시연]

> 0.1초 만에 자연스러운 나레이션이 뚝딱 만들어집니다. 성우 섭외 안 해도 돼요.

> 실시간 고객 응대 챗봇에도 이상적이에요. 100밀리초 지연이면 실시간 대화에서 전혀 어색하지 않거든요.

[자막: 활용처 - 교육 콘텐츠 / 고객 응대 / 오디오북 / 접근성 서비스]

> 특히 다국어 서비스를 운영하시는 분들, 15개 언어 음성을 하나의 모델로 처리할 수 있다는 건 엄청난 장점입니다.

## [본론3 - 2분] Denovo

[화면: "03 Denovo" 타이틀 카드]
[화면: Denovo 서비스 메인 화면 캡처]

> 세 번째 도구는 제가 개인적으로 가장 충격받은 건데요. 바로 Denovo입니다.

> Denovo는 한마디로 이겁니다.

[자막: "아이디어를 입력하면, AI가 스타트업을 통째로 만들어준다"]

> 네, 제대로 들으신 겁니다. 여러분이 사업 아이디어를 텍스트로 입력하면, Denovo가 자율적으로 스타트업을 구축해줍니다.

[화면: Denovo 아이디어 입력 화면]

> 비즈니스 모델 설계, 시장 조사, 기술 스택 선정, 심지어 코드 생성과 배포까지. 사람이 개입하지 않아도 AI가 알아서 척척 진행합니다.

### 데모 시나리오

[화면: Denovo에 아이디어를 입력하는 과정 시연]

> 예를 들어 이런 아이디어를 넣어볼게요.

[자막: "반려동물 건강 모니터링 구독 서비스"]

> 이걸 입력하면 Denovo가 뭘 하느냐.

[화면: Denovo 작업 진행 과정 단계별 애니메이션]

> 먼저 시장을 분석하고, 경쟁사를 조사하고, 비즈니스 모델을 수립합니다. 그다음 필요한 기능을 정의하고, 실제 웹 서비스의 코드를 작성하고, 배포 가능한 상태까지 만들어 줍니다.

[자막: 시장분석 -> 비즈니스 모델 -> 기능 정의 -> 코드 생성 -> 배포]

> 물론 이걸로 바로 유니콘 기업이 되는 건 아니겠지만요, 아이디어 검증 단계에서 프로토타입을 빠르게 만들어본다는 관점에서 보면 정말 혁명적인 도구예요.

> 비개발자분들도 자기 아이디어를 실제 서비스로 만들어볼 수 있다는 거잖아요. 진입장벽이 확 낮아지는 거죠.

[자막: 비개발자도 아이디어만으로 서비스 프로토타입 제작 가능]

## [비교/정리 - 1분]

[화면: 3가지 도구 비교표 인포그래픽]

> 자, 그러면 오늘 소개한 세 가지 도구를 한눈에 정리해 볼게요.

[화면: 비교표 테이블 애니메이션]

| 구분 | Claude Code Voice | Lightning V3 | Denovo |
|------|------------------|--------------|--------|
| 핵심 기능 | 음성 기반 AI 코딩 | 초고속 TTS | AI 스타트업 빌더 |
| 주요 대상 | 개발자 | 콘텐츠 제작자 / 서비스 개발자 | 창업자 / 기획자 |
| 강점 | 핸즈프리, 멀티태스킹 | 100ms 지연, 15개+ 언어 | 아이디어만으로 서비스 구축 |
| 활용 장면 | 코딩, 브레인스토밍 | 나레이션, 챗봇, 다국어 서비스 | 아이디어 검증, 프로토타이핑 |

> Claude Code Voice는 개발자의 생산성을 극대화해주고요, Lightning V3는 음성 합성이 필요한 모든 곳에서 최강이고요, Denovo는 아이디어를 실현하는 속도를 비약적으로 높여줍니다.

[자막: 공통점 - AI가 인간의 실행 속도를 비약적으로 높여주는 도구들]

> 이 세 가지 도구의 공통점이 뭘까요? 바로 "생각에서 실행까지의 거리"를 극단적으로 줄여준다는 겁니다. 음성으로 말하면 코드가 되고, 텍스트를 넣으면 목소리가 되고, 아이디어를 쓰면 서비스가 됩니다.
