## 1. 플랫폼 개요

### 1-1. GEO-AIO란?

GEO-AIO는 **AI 검색엔진(AI Overview, Generative Engine)에 최적화된 콘텐츠를 분석하고 생성하는 올인원 도구**이다.

기존 SEO가 Google 키워드 순위에 집중했다면, GEO-AIO는 ChatGPT, Gemini, Perplexity, Google AI Overview, Microsoft Copilot 등 **AI가 콘텐츠를 인용하고 답변에 포함시키는 것**에 초점을 맞춘다.

- **GEO (Generative Engine Optimization):** AI 검색엔진에서 콘텐츠가 선택·인용되도록 최적화
- **AIO (AI Overview):** Google AI Overview에 노출될 확률을 높이는 최적화

> **핵심 비전:** "AI 검색 시대, 새로운 콘텐츠 전략" — Claude AI를 활용하여 콘텐츠의 GEO/AIO 점수를 분석하고, 최적화된 콘텐츠를 자동으로 생성

### 1-2. 왜 GEO/AIO가 필요한가?

| 지표 | 수치 |
|------|------|
| Google 검색에서 AI Overview 표시 비율 | **40%+** |
| 소비자 중 AI 검색 사용 비율 | **68%** |
| 블로그 클릭률(CTR) 하락폭 | **최대 40%** |
| AI에 무시되는 콘텐츠 비율 | **90%** |

### 1-3. AI가 인용하는 3가지 조건

1. **구조화된 답변** — FAQ, How-to, 리스트 형식 활용
2. **E-E-A-T** — 경험(Experience), 전문성(Expertise), 권위성(Authoritativeness), 신뢰성(Trustworthiness)
3. **원본 데이터** — 고유 통계, 사례, 인사이트

### 1-4. 지원 AI 검색엔진

- Google AI Overview
- ChatGPT / SearchGPT
- Google Gemini
- Perplexity AI
- Microsoft Copilot

### 1-5. 법적 보호

특허 및 저작권 등록을 통해 법적 보호를 받고 있음.

## 4. 콘텐츠 분석 시스템

### 4-1. 분석 입력

- **타겟 키워드** (선택) — 예: "AI 콘텐츠 최적화"

- **분석할 콘텐츠** (필수) — 최소 50자 이상 권장

### 4-2. 분석 결과 5개 탭

| 탭 | 내용 |
|----|------|
| **종합 개요** | 종합 점수, AIO 점수, GEO 점수 요약 |
| **AIO 분석** | 구조화된 답변, 명확성/간결성, 인용 가능성, AI Overview 노출 확률 |
| **GEO 분석** | AI 검색 친화도, 구조화 데이터, 의미적 완성도, E-E-A-T 신호 |
| **키워드 분석** | 키워드 밀도, 관련 키워드, LSI 키워드 |
| **개선 제안** | 구체적 개선 사항 및 AI 최적화 변환 |

### 4-3. AIO 분석 4대 지표

| 지표 | 설명 |
|------|------|
| 구조화된 답변 (structuredAnswer) | FAQ, How-to, 리스트 형식 활용도 |
| 명확성/간결성 (clarity) | 콘텐츠의 이해도와 간결함 |
| 인용 가능성 (citability) | AI가 이 콘텐츠를 인용할 확률 |

## 11. Make.com 자동화 연동

### 11-1. 개요

Make.com은 프로그래밍 없이 다양한 앱과 서비스를 연결하는 노코드 자동화 도구 (전 세계 50만+ 기업 사용).

**웹훅 URL:** `https://aio-geo-optimizer.vercel.app/api/webhook`

### 11-2. 6가지 웹훅 액션

| 액션 | 설명 |
|------|------|
| `generate` | 콘텐츠 생성 |
| `analyze` | 콘텐츠 분석 |
| `optimize` | 콘텐츠 최적화 |
| `convert` | SNS 채널 변환 |
| `keyword-analysis` | 키워드 경쟁 분석 |
| `generate-series` | 시리즈 기획 |

### 11-3. 4가지 자동화 시나리오

| 난이도 | 시나리오 | 소요 시간 |
|--------|---------|----------|
| 초급 | 키워드 → 콘텐츠 자동 생성 → Google Sheets 저장 | 15분 |
| 중급 | 콘텐츠 생성 → 멀티채널 자동 변환 → SNS 게시 | 30분 |
| 중급 | 정기 시리즈 자동 기획 → 매일 1편 발행 | 30분 |
| 고급 | 경쟁사 모니터링 → 대응 콘텐츠 자동 생성 | 45분 |

### 11-4. 다운로드 블루프린트

- `blog-auto-generate.json` — 블로그 자동 생성
- `content-analyze-optimize.json` — 콘텐츠 분석 + 최적화
- `multi-channel-distribute.json` — 멀티채널 배포
